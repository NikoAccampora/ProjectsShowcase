Um projeto de criação de website, inspirado na série de jogos 'Mega Man'. Inclui uma página 'Home' e mais três páginas diferentes.

DEPENDÊNCIAS: MySQL Driver; Spring Web; Spring Boot DevTools; Thymeleaf; Spring Data JPA

Caso não esteja familiarizado(a) com MySQL, aínda é possível utilizar os dados dos arquivos 'home.html', 'characters.html', 'games.html' e 'weapons.html' para reproduzir os resultados utilizando qualquer software de criação de website que preferir.
